Kebutuhan Sistem untuk Membuat Aplikasi

Bagian ini berisi software dan langkah awal yang perlu disiapkan sebelum mulai membuat aplikasi.
1. Sistem operasi yang direkomendasikan: Windows 10/11 64-bit atau macOS versi terbaru. 2. Software utama yang perlu diinstal: - Visual Studio Code untuk menulis dan mengelola kode. - XAMPP untuk menjalankan MySQL secara lokal. - Node.js LTS untuk menjalankan backend Node.js. - Browser Google Chrome untuk testing aplikasi. 3. Tutorial instal Visual Studio Code: - Buka website resmi https://code.visualstudio.com/ - Klik Download sesuai sistem operasi Anda. - Jalankan file installer yang sudah diunduh. - Klik Next sampai selesai, lalu centang opsi menambahkan VS Code ke PATH jika tersedia. - Setelah selesai, buka VS Code dan install extension dasar seperti: - Prettier - ESLint - Vue Language Features 4. Tutorial instal XAMPP: - Buka website resmi https://www.apachefriends.org/ - Download XAMPP sesuai sistem operasi Anda. - Jalankan installer lalu pilih komponen minimal Apache dan MySQL. - Selesaikan proses instalasi. - Buka XAMPP Control Panel. - Jalankan MySQL. - Buat database baru melalui phpMyAdmin atau tool database lain sesuai nama database yang diperlukan sistem. 5. Tutorial instal Node.js: - Buka website resmi https://nodejs.org/ - Download versi LTS. - Jalankan installer sampai selesai. - Setelah terpasang, cek di terminal dengan perintah: - node -v - npm -v 6. Kebutuhan minimum perangkat: - RAM minimal 2 GB - Storage kosong minimal 10 GB - Koneksi internet stabil untuk download package dan dokumentasi 7. Setelah semua siap: - Buka project di VS Code - Jalankan MySQL dari XAMPP - Buat database sesuai nama yang diminta aplikasi - Jalankan npm install - Jalankan server sesuai instruksi project
